# Design Brief — Life Admin Tracker

| Aspect | Direction |
|--------|-----------|
| **Purpose** | Productivity dashboard for college students tracking critical life admin tasks (bills, renewals, insurance) |
| **Tone** | Professional-minimal: quiet confidence, respects user time, eliminates unnecessary decoration |
| **Palette** | Cream off-white base (L:0.98), muted teal-blue primary (L:0.45, H:252°), semantic urgency colors: warm red for overdue (H:25°), warm orange for due-this-week (H:64°), cool blue for due-later (H:268°) |
| **Typography** | DM Sans (display, geometric-friendly), Inter (body, dashboard standard), monospace for timestamps |
| **Radius** | 8px (friendly, professional consistency) |
| **Elevation** | Subtle 2px card shadow for layering without drama; 4px elevated shadow for modals/popovers |
| **Spacing** | Generous gaps (8px, 16px, 24px) emphasizing breathability over density |
| **Structural Zones** | Header with bottom border, content cards on muted background, footer minimal; alternating card surfaces for rhythm |
| **Urgency UI** | Left border accent (destructive/warning/accent colors) on task cards; semantic color use in summary metrics |
| **Motion** | Smooth transitions (0.3s cubic-bezier) on hover/focus; no entrance animations (fast load) |
| **Constraints** | No decoration, no gradients, no unnecessary shadows; high contrast text (AA+ compliant) |
| **Signature Detail** | Multi-urgency color-coded task cards with left-border accent; clean summary-card metric display showing counts with at-a-glance status |
